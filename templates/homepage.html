MediScan AI - an all in one medical solution

MediScan AI bundles multiple disease detection models so clinicians and users can try several imaging-based diagnostics from a single, easy-to-use web interface. The site collects specialist models for Covid, Brain Tumor, Breast Cancer, Alzheimer, Diabetes, Pneumonia, and Heart Disease detection and presents them under one roof.

AI in HealthCare

I started this project after building an initial breast cancer prediction model â€” it was a small, rewarding experiment that taught me a lot about data, preprocessing, and model behaviour. After seeing promising results and enjoying the process, I decided to expand that work into a broader toolkit so more conditions could be explored from medical images. MediScan AI is my practical attempt to package those experiments into a usable demo that others can try and learn from.

Machine Learning

My interest in machine learning grew while iterating on model ideas and evaluation strategies for the breast cancer project. I learned how small design choices affect real-world performance, so I embraced classical supervised learning techniques and practical validation workflows. That hands-on experience convinced me to apply ML across several imaging tasks and to build a simple web interface where these models can be explored quickly.

Natural Language Processing

While my main focus is imaging, I also explored NLP because clinical text unlocks a different set of signals. I wanted to see how notes and reports could complement image-based models, so I experimented with basic text pipelines to extract structured information from unstructured records. These experiments helped me appreciate how combining modalities can improve clinical workflows and informed future directions for MediScan AI.
